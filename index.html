请求参数 ​

参数 值 可选 说明
type 见后表 是 句子类型
encode 见后表 是 返回编码
charset 见后表 是 字符集
content 见后表 是 返回数据

句子类型（参数） ​

参数 说明
Anime 动画
Comic 漫画
Game 游戏
Literature 文学
Original 原创
Internet 来自网络
Other 其他
Video 影视
Poem 诗词
NCM 网易云
Philosophy 哲学
Funny 抖机灵
All 全部

返回编码（参数） ​

参数 说明
text 返回纯洁文本
json 返回格式化后的 JSON 文本
其他 返回格式化后的 JSON 文本

字符集（参数） ​

参数 说明
utf-8 返回 utf-8 编码的内容
gbk 返回 gbk 编码的内容。不支持与异步函数同用
其他 返回 utf-8 编码的内容

返回信息 ​

返回参数名称 描述
id 一言标识
yiyan 一言正文。编码方式 unicode。使用 utf-8。
type 类型。请参考第三节参数的表格
from 一言的出处
from_who 一言的作者
creator 添加者
creator_uid 添加者用户标识
reviewer 审核员标识
uuid 一言唯一标识；可以链接到 https://txt.201562.xyz/?uuid= 查看这个一言的完整信息
commit_from 提交方式
created_at 添加时间
length 句子长度
示例：https://txt.201562.xyz/?type=Anime@Game&encode=text
